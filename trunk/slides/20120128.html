What is StackDrom?

  • An effort to simplify the path: idea ⇒ design flow
  • Single processor description format to be used everywere
  • Initial set of tools using this format
  • Couple of examples

Processor description format

  • JSON based format
  • ISA from scratch = NO binary compatibility
  • multi-format instructions
  • configurability / scalability
  • 1+ stack architecture
  • FORTH instruction mnemonic
  • examples: J1, Sputnik ↵

Documentation

  • StackDrom.js = documentation automation script
  • istruction set summary
  • istruction tables
  • cross-reference
  • examples: J1, Sputnik ↵

Processor model

  • Perl library for Verilog RTL generation
  • set of main units and flexible interconnect
    • instruction prefetch buffer ⇒
    • instruction decoder
    • 1+ stack cache ⇒
    • 1+ ALU
    • Load / Store + write buffer
  • In progress...
